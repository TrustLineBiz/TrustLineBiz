# Skills — Reusable Runbooks

Each file is a self-contained playbook for a specific recurring task.
Reference these when asking Claude to perform that type of work.

| Skill              | File                  | Use When                                      |
|--------------------|-----------------------|-----------------------------------------------|
| Copywriting        | `copywriting.md`      | Updating headlines, CTAs, body copy           |
| SEO                | `seo.md`              | Improving rankings, meta, structured data     |
| Design Tokens      | `design-tokens.md`    | Adding UI, checking brand consistency         |
| Forms              | `forms.md`            | Adding/changing form fields or behavior       |
| Security           | `security.md`         | Security review, headers, input handling      |
| Deploy             | `deploy.md`           | Deploying to Netlify, checking live site      |
